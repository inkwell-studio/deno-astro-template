# Deno & Astro Template

This is an [Astro](https://astro.build/) site that runs on [Deno Deploy](https://deno.com/deploy). It is configured to use
[Tailwind CSS](https://tailwindcss.com/).

It is available at [https://deno-astro-template-i1.deno.dev/](https://deno-astro-template-i1.deno.dev/).

## Development

[Deno 2](https://deno.com/) is necessary.

Using [Visual Studio Code](https://code.visualstudio.com/) is optional, though this project is configured for it.

See [DEVELOPMENT.md](./DEVELOPMENT.md) for detailed instructions.
